###Las configuraciones basicas que debo hacer en laravel:
1) configurar la zona horario CONFIGURAR IDIOMA Y USO HORARIO EN EL ARCHIVO APP (config/app.php)
2) configurar el idioma // RUTA PARA CAMBIAR IDIOMA resources\lang\es copiar el repo es
3) Configurar la base de datos

### Comandos basicos 
-php artisan serve // Inicia el servidor 

-php artisan app:name NOMBRE // cambia el nombre a la aplicacion 

-php artisan make:controller // crea un controlador

-php artisan make:controller PhotoController --resource //crea un controlador completo con su crud 

-php artisan make:model Cliente -m// crea un modelo NOTA El nombre debe empezar en mayuscula y esta en singular -m crea su migracion para la db

-php artisan migrate // Migra lo que tengasmos en el archivo de migraciones a la db

-php artisan make:migration add_slug_clientes --table=clientes // crea un campo nuevo en una tabla
